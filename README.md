# 🍔 Food Ordering App

A simple food ordering application built using the **MERN stack** (MongoDB, Express.js, React, Node.js).

## 🛠 Tech Stack

- **Frontend:** React
- **Backend:** Node.js + Express
- **Database:** MongoDB

## 📦 Features

- User registration and login
- View list of available food items
- Add items to cart
- Place orders
